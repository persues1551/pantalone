# 宏观分析研究模板

## 研究流程

### Phase 1: 宏观数据收集
```bash
# GDP、CPI、PMI等
python3 -c "
import akshare as ak
# PMI
df = ak.macro_china_pmi()
print(df.tail(3))
# CPI
df2 = ak.macro_china_cpi()
print(df2.tail(3))
# 社融
df3 = ak.macro_china_shrzgm()
print(df3.tail(3))
"
```

### Phase 2: 政策分析
- 央行货币政策（MLF/LPR/降准降息）
- 财政政策（专项债/减税）
- 产业政策（新能源/AI/半导体）
- 监管政策（IPO/再融资/量化）

### Phase 3: 资金面分析
- 北向资金流向
- 融资融券余额
- 新基金发行规模
- 产业资本增减持

### Phase 4: 全球联动
- 美股/美债/美元指数
- 大宗商品（原油/铜/黄金）
- 汇率（人民币/日元）
- 地缘政治事件

### Phase 5: 假设生成与评审
