# 财报分析研究模板

## 研究流程

### Phase 1: 财报数据收集
```bash
# 获取个股财务数据
python3 -c "
import akshare as ak
# 利润表
df = ak.stock_financial_report_sina(stock='600519', symbol='利润表')
print(df.head(4))
# 资产负债表
df2 = ak.stock_financial_report_sina(stock='600519', symbol='资产负债表')
print(df2.head(4))
"
```

### Phase 2: 关键指标计算
- 营收增长率（同比/环比）
- 净利润增长率
- 毛利率/净利率变化
- ROE趋势
- 现金流状况
- 资产负债率

### Phase 3: 行业对比
与同板块其他公司对比：
- 估值（PE/PB/PS）对比
- 盈利能力对比
- 成长性对比

### Phase 4: 假设生成
基于财报数据生成投资假设，例如：
- "XX公司Q1营收超预期30%，但市场仅反应5%，存在预期差"
- "YY公司毛利率连续3季度下滑，竞争格局恶化"

### Phase 5: 评审与报告
