# amadeus_watcher.py 盘中异动监控脚本

## 功能
实时监控观察池标的，触发条件时输出异动提醒。

## 触发条件
| 条件 | 阈值 | 优先级 |
|------|------|--------|
| 涨幅异动 | ≥5% | 高 |
| 跌幅异动 | ≤-3% | 紧急 |
| 涨停/跌停 | ±10%/±20% | 紧急 |
| 止损预警 | 浮亏接近止损线(差1%) | 紧急 |
| 止损触发 | 浮亏超过止损线 | 紧急 |

## 使用
```bash
python3 ~/.hermes/scripts/amadeus/amadeus_watcher.py
```

## Cron配置
```yaml
schedule: "*/30 9-11,13-15 * * 1-5"  # 盘中每30分钟
job_id: 2695c52bd0bf
```

## 输出格式
```
🔔 异动监控 | 2026-05-21 10:49
监控标的: 94只 | 触发: 39条
==================================================
🚨 利仁科技 001259 | 涨停
   价格: 88.58 | 涨跌: +10.00%
   涨停 +10.00%
```

## 无触发时
```
✅ 异动监控 | 2026-05-21 10:49 | 监控94只，无异常
```

## 注意事项
- 上午无触发时不发送（避免噪音）
- 只有涨停/跌停/止损预警才触发发送
- 读取session_context.json中的stock_notes获取观察池
