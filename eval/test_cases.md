## 情绪温度计算测试用例

### 测试用例 1：正常市场
```
输入：
- 涨停：80只，跌停：5只
- 上涨：3000只，下跌：2000只
- 最高连板：8连板
- 炸板：15只，涨停池：95只
- 成交额：3.0万亿，前日：2.8万亿

期望输出：
- 涨停跌停比：80/5=16 → 16/20*25=20分
- 市场宽度：3000/(3000+2000)=60% → 60/80*25=18.75分
- 连板高度：8连板 → 8/15*20=10.67分
- 炸板率：15/(95+15)=13.6% → (1-0.136)*15=12.96分
- 成交额变化：(3.0-2.8)/2.8=7.1% → 7.1/20*15=5.33分
- 情绪温度：20+18.75+10.67+12.96+5.33=67.71 → 高潮

验证：情绪温度在61-80区间，判定为"高潮"
```

### 测试用例 2：冰点市场
```
输入：
- 涨停：20只，跌停：80只
- 上涨：1000只，下跌：4000只
- 最高连板：3连板
- 炸板：30只，涨停池：50只
- 成交额：1.5万亿，前日：2.0万亿

期望输出：
- 涨停跌停比：20/80=0.25 → 0.25/20*25=0.31分
- 市场宽度：1000/(1000+4000)=20% → 20/80*25=6.25分
- 连板高度：3连板 → 3/15*20=4分
- 炸板率：30/(50+30)=37.5% → (1-0.375)*15=9.38分
- 成交额变化：(1.5-2.0)/2.0=-25% → 0分
- 情绪温度：0.31+6.25+4+9.38+0=19.94 → 冰点

验证：情绪温度在0-20区间，判定为"冰点"
```

## 排雷检查测试用例

### 测试用例 1：正常股票
```
输入：
- ST标记：无

- 审计意见：标准无保留
- 经营现金流：连续3年正

```

```

- 经营现金流：连续2年负
```
