# scoring.md — 评分标准

## 综合评分公式（6维度100分制）

评分公式：预案命中率 + 执行纪律 + 资金流判断 + 情绪判断 + 题材判断 + 预测准确率 = 综合 xx/100

| 维度 | 权重 | 满分 | 对应段号 | 评判标准 |
|------|------|------|---------|---------|
| 预案命中率 | 20% | 20分 | 三（预案验证） | 早盘预案 vs 实际走势命中率 |
| 执行纪律 | 15% | 15分 | 四（观察股验证） | 是否按计划操作，有无情绪化交易 |
| 资金流判断 | 15% | 15分 | 二（板块资金流） | 板块资金流入方向判断准确度 |
| 情绪判断 | 15% | 15分 | 七（情绪与连板） | 情绪温度判断 vs 次日实际走势验证 |
| 题材判断 | 15% | 15分 | 八（题材轮动） | 主线研判 vs 次日实际领涨/领跌板块 |
| 预测准确率 | 20% | 20分 | 九 vs 十 | 明日预测 vs 昨日验证准确率 |

## 各维度评分细则

### 预案命中率（20分）

| 命中情况 | 得分 |
|----------|------|
| 全部命中 | 20分 |
| 部分命中（1-2个方向） | 12-16分 |
| 少量命中（1个方向） | 8-12分 |
| 完全未命中 | 0-8分 |

### 执行纪律（15分）

| 纪律情况 | 得分 |
|----------|------|
| 完全按计划执行 | 15分 |
| 基本按计划，有小偏差 | 10-14分 |
| 部分违反纪律 | 5-10分 |
| 严重违反纪律（追高/恐慌割肉） | 0-5分 |

### 资金流判断（15分）

| 判断情况 | 得分 |
|----------|------|
| 资金主线方向完全正确 | 15分 |
| 资金主线方向部分正确 | 10-14分 |
| 资金主线方向有偏差 | 5-10分 |
| 资金主线方向完全错误 | 0-5分 |

### 情绪判断（15分）

| 判断情况 | 得分 |
|----------|------|
| 情绪温度判断准确（偏差<1级） | 15分 |
| 情绪温度偏差1级 | 11-14分 |
| 情绪温度偏差2级 | 5-10分 |
| 情绪温度偏差≥3级 | 0-5分 |

### 题材判断（15分）

| 判断情况 | 得分 |
|----------|------|
| 主线方向完全正确 | 15分 |
| 主线方向部分正确 | 10-14分 |
| 主线方向有偏差 | 5-10分 |
| 主线方向完全错误 | 0-5分 |

### 预测准确率（20分）

| 准确情况 | 得分 |
|----------|------|
| 5/5 正确 | 20分 |
| 4/5 正确 | 16分 |
| 3/5 正确 | 12分 |
| 2/5 正确 | 8分 |
| 1/5 正确 | 4分 |
| 0/5 正确 | 0分 |

## 情绪温度分级

| 分级 | 分数 | 含义 | 操作建议 |
|------|------|------|----------|
| 冰点 | 0-20 | 10%机会 | 寻找超跌反弹机会 |
| 修复 | 21-40 | 30%机会 | 关注率先反弹的板块 |
| 升温 | 41-60 | 60%机会 | 主攻方向确认 |
| 高潮 | 61-80 | 40%风险 | 谨慎追高，注意分歧 |
| 退潮 | 81-100 | 20%风险 | 降低仓位，等待新主线 |

## 预测准确率计算

**计算公式**：正确预测数 / 总预测数 × 100%

**连续错误标记规则**：
- 某维度连续3次错误 → 自动标记【该维度判断逻辑待调整】
- 写入复盘「八、失败点」
- 必须在下次复盘中修正该维度的判断逻辑

## 评分输出格式

```
## 综合评分

| 维度 | 权重 | 得分 | 说明 |
|------|------|------|------|
| 预案命中率 | 20% | xx | 扣分原因 |
| 执行纪律 | 15% | xx | 扣分原因 |
| 资金流判断 | 15% | xx | 扣分原因 |
| 情绪判断 | 15% | xx | 扣分原因 |
| 题材判断 | 15% | xx | 扣分原因 |
| 预测准确率 | 20% | xx | 扣分原因 |
| **综合** | **100%** | **xx/100** | **等级** |

**成功点**：3-5条
**失败点**：3-5条
**明日优化**：1-3条
```

## 等级划分

| 等级 | 分数 | 含义 |
|------|------|------|
| 优秀 | 90-100 | 表现优异，继续保持 |
| 良好 | 80-89 | 表现不错，有改进空间 |
| 中等 | 70-79 | 表现一般，需要改进 |
| 较差 | 60-69 | 表现不佳，重点改进 |
| 很差 | <60 | 表现糟糕，全面复盘 |
