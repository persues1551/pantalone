# 观察池自动扫描配置 v1.2

## 核心理念（来自知乎职业投资人）

**财报季选股法**：
1. 每年1月、4月、7月、10月是财报季
2. 看业绩大增公司的财报、招股说明书
3. 关注关键词：产品供不应求、行业高景气度上行、市场超预期拓展、新品上市持续超预期、产品价格中枢持续上涨、供给偏紧、需求旺盛
4. 满足"业绩大增+关键词"→ 重点关注

## 扫描策略

### 1. 财报扫描（新增）
- **时间**：财报季（1/4/7/10月）每日
- **范围**：业绩预告/快报
- **条件**：业绩大增 + 关键词命中
- **入池**：C池，验证期**1天**

### 2. ML信号扫描（每日）
- **时间**：每天16:00
- **范围**：观察池活跃股票
- **条件**：ML评分>60，信号为"买入"
- **入池**：C池，验证期**1天**

### 3. 行业轮动扫描（每日）
- **时间**：每天16:30
- **范围**：资金流入前3行业
- **条件**：行业龙头，市值前3
- **入池**：C池，验证期**1天**

### 4. 新闻热点扫描（每日）
- **时间**：每天17:00
- **范围**：当日新闻热点
- **条件**：有股票代码提及，优先关键词命中
- **入池**：C池，验证期**1天**

### 5. 验证检查（每日）
- **时间**：每天18:00
- **范围**：所有观察池股票
- **动作**：检查验证日期，提醒到期股票

## 验证节奏

**短线逻辑**：验证期1天
- 今天入池 → 明天验证
- 快速验证 → 快速决策
- 逻辑兑现 → 升级/持有
- 逻辑证伪 → 退池

## 过滤规则

### 排除条件
1. 已在池中（非退池状态）
2. ST股、*ST股
3. 停牌股

### 优先条件
1. 财报关键词命中优先
2. ML评分>70优先
3. 行业龙头优先
