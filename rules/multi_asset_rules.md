# multi_asset_rules.md — 多资产规则

## 版本定位与优先级

1. 股票/A股相关规则完全沿用原框架，优先级高于多资产规则
2. 不启用 token 节省规则
3. 不压缩股票观察股、财报分析、情绪温度、模拟盘验证
4. 股票报告仍按原要求执行：数据核验 → 情绪温度 → 题材强度 → 基本面评级 → 技术触发 → 模拟盘验证 → 复盘迭代
5. 每次股票类报告仍必须给出10只以上观察股，并保持详细分析
6. 多资产内容只作为额外补充模块，不替代股票研究框架
7. **触发条件**：用户明确要求「多资产早报 / 多资产分析 / 资产配置 / 债券 / 黄金 / 汇率 / 商品」时，才展开多资产模块
8. 普通「今日早报、盘前情报、午盘总结、收盘复盘」仍以股票/A股为主，多资产只写精简补充

## 多资产分析公式

宏观周期定位 + 大类资产比较 + 单类资产基本面 + 技术择时 + 风险预算 + 模拟验证 = 多资产投资系统

**分析顺序**：数据核验 → 宏观环境 → 利率与流动性 → 通胀与增长 → 大类资产强弱 → 单类资产分析 → 风险提示 → 模拟观察 → 复盘迭代

## 多资产数据铁律

禁止编造数据。查不到就写「数据缺失」。多源冲突写【数据存疑】。关键数据缺失时，不生成模拟计划。

所有多资产数据必须尽量标注：日期、口径、来源、收盘/盘中/历史数据。

## 宏观环境简表

| 周期 | 特征 | 利好资产 | 利空资产 |
|------|------|----------|----------|
| 复苏 | 增长改善，通胀温和 | 股票、商品 | 债券中性 |
| 过热 | 增长强，通胀高 | 商品 | 债券承压 |
| 滞胀 | 增长弱，通胀高 | 黄金/部分商品 | 股票承压 |
| 衰退 | 增长弱，通胀回落 | 债券 | 股票盈利承压 |
| 宽松流动性 | 利率下行，风险偏好提升 | 成长股、长久期资产、黄金 | — |
| 紧缩流动性 | 利率上行，美元走强 | — | 股票估值、长久期资产 |

## 债券分析模块

**分析对象**：利率债、信用债、可转债、城投债、产业债、债券基金/ETF

**核心变量**：到期收益率、久期、信用评级、信用利差、期限利差、违约风险、流动性、央行政策、通胀预期

**规则**：
- 利率下行 → 长久期债券价格通常受益
- 利率上行 → 长久期债券价格通常承压
- 信用利差扩大 → 信用风险上升；收窄 → 风险偏好改善
- 低评级高收益债不得包装成稳健资产
- 可转债必须同时看债性、股性、转股溢价率、正股质量

## 货币/汇率分析模块

**关注变量**：美元指数、人民币汇率、中美利差、实际利率、央行政策、外储变化、贸易顺差/逆差、资本流动、避险情绪

**规则**：
- 美元走强 → 通常压制新兴市场资产、黄金和部分商品
- 人民币贬值压力上升 → 关注外资流动、出口链、进口成本、航空等外债压力行业
- 人民币升值 → 可能提升人民币资产风险偏好，利于外资流入
- 汇率不做单点预测，只做区间、压力和影响判断

## 黄金分析模块

**四个核心变量**：实际利率、美元指数、避险需求、央行购金

**规则**：
- 实际利率下行 → 黄金通常受益
- 美元走弱 → 黄金通常受益
- 地缘风险上升 → 黄金避险属性增强
- 央行持续购金 → 提升长期配置逻辑
- 实际利率上行且风险偏好强 → 黄金可能承压

**资产分类**：实物黄金、黄金ETF、黄金股票、黄金期货、黄金基金

**注意**：黄金股票不是黄金本身，还要看公司成本、产量、储量、财务杠杆。

## 商品分析模块

**分类**：能源（原油/天然气/煤炭）、贵金属（黄金/白银）、工业金属（铜/铝/锌/镍）、黑色系（铁矿/螺纹钢/焦煤/焦炭）、农产品（大豆/玉米/棉花/白糖）

**核心变量**：供给、需求、库存、产能、成本曲线、季节性、地缘风险、美元指数、中国需求、全球制造周期

**规则**：
- 库存下降 + 需求改善 = 偏强
- 库存上升 + 需求转弱 = 偏弱
- 供给冲击可短期推升价格，但长期看需求与库存
- 商品波动大，不得描述为低风险资产

## 基金/ETF分析模块

**必看指标**：底层资产、跟踪指数、规模、成交额、费率、跟踪误差、折溢价、持仓集中度、风格暴露、历史回撤

**分类**：宽基ETF、行业ETF、债券ETF、黄金ETF、商品ETF、QDII基金、主动权益基金、货币基金

**规则**：
- ETF不是风险更低，而是风险更透明
- 行业ETF仍有行业周期风险
- QDII额外看汇率、海外市场、额度、溢价
- 货币基金看流动性和收益率，不幻想高收益

## 大类资产配置观察

**资产分类**：
- 进攻资产：股票、成长ETF、商品周期
- 防守资产：债券、现金、红利资产
- 对冲资产：黄金、美元资产
- 流动性资产：货币基金、短债

**组合观察必须包含**：当前强弱、相关性、波动风险、最大回撤风险、流动性、仓位上限、失效条件

## 多资产模拟仓位规则

模拟盘仍为 200,000 元。

| 风险等级 | 资产类型 | 单类上限 |
|----------|----------|----------|
| 低风险 | 货币基金、短债ETF、短久期利率债 | 40% |
| 中风险 | 宽基ETF、高等级债券ETF、黄金ETF | 25% |
| 高风险 | 个股、行业ETF、商品ETF、可转债 | 15% |
| 极高风险 | 高波动商品、低评级信用债、杠杆ETF | 默认不进入 |

**约束**：单一资产≤总资金20%、单一行业≤30%、高波动资产总仓位≤30%。情绪高潮时权益类仓位下降。流动性紧缩时长久期和高估值资产降权。

## 多资产观察对象规则

当用户明确要求多资产分析时，观察对象**不少于10个**，且至少覆盖3类资产。

**默认结构**：股票/ETF 4-6个、债券/债基 2-3个、黄金/商品 2-3个、货币/现金管理 1-2个

每个观察对象包含：代码/名称、资产类别、观察理由、核心数据、风险点、触发条件、失效条件、是否允许进入模拟盘

## 多资产评级

| 评级 | 含义 |
|------|------|
| A | 基本面强、流动性好、风险收益比高、数据完整 |
| B | 逻辑较强，但存在估值、利率、周期或波动风险 |
| C | 可观察，但只适合小仓位或等待触发 |
| D | 逻辑弱、风险高、数据不完整或趋势破坏 |
| E | 高风险、财务/信用/流动性存疑，默认回避 |

## 多资产数据缺失处理

- 股票缺财报 → 不给A/B评级
- 债券缺收益率/久期/信用评级 → 不进模拟
- 黄金缺美元/实际利率/价格 → 不生成明确判断
- 商品缺库存/供需/价格 → 只观察
- ETF缺规模/成交额/折溢价 → 不进模拟

## 普通股票报告中的多资产补充

当用户说「今日早报 / 盘前情报 / 早盘预案 / 午盘总结 / 收盘复盘」时，仍以股票/A股流程为主，在报告末尾增加简短模块：

```
【多资产补充】
1. 债券：一句话说明利率/债市对权益估值的影响。
2. 汇率：一句话说明人民币、美元指数对A股外资和风险偏好的影响。
3. 黄金：一句话说明避险与实际利率变化。
4. 商品：一句话说明原油、铜、煤炭等对相关板块的影响。
5. 结论：多资产环境对今日A股是加分、中性还是减分。
```

该模块保持简洁，不影响股票主报告完整性。

## 多资产禁止行为

1. 不得把高收益债说成稳健理财
2. 不得把黄金说成必涨避险
3. 不得把债券说成无风险
4. 不得忽略久期风险
5. 不得忽略汇率风险
6. 不得忽略商品高波动
7. 不得用股票逻辑分析债券
8. 不得用单一消息判断大类资产
9. 不得生成真实交易建议
10. 不得承诺收益

## 多资产最终纪律

- 股票研究能力不变，股票规则优先级高于多资产补充规则
- 多资产模块只做扩展，不削弱原A股流程
- 先核验数据，再判断资产；先判断风险，再讨论机会；先做组合，再看单品种
- 宁可写数据缺失，也不能编造
