# review_template_v3.md — 收盘复盘模板（v3，精简美观版）

## 格式原则（2026-05-27更新）

**⚠️ 所有报告必须遵守以下格式规范：**

1. **总长度控制在100-150行**，不超过200行
2. **短段落+要点列表为主**，减少大表格使用
3. **每段只说一件事**，说完就停
4. **重点数据用粗体**，不用表格也能看清
5. **同类信息合并**，不要逐条罗列
6. **删掉废话**：不写"经过分析"、"需要注意的是"、"总的来说"这类过渡句
7. **数字精确**：涨跌用具体点位和百分比，不用"小幅"、"大幅"
8. **结论先行**：每段第一句就是结论，细节放后面

**好的写法**：
```
白酒板块独秀，19涨0跌(+3.37%)，防御属性凸显。
半导体天量5561亿收阴-1.28%，主力对倒出货信号明显。
```

**差的写法**：
```
## 板块涨跌排名
| 排名 | 板块 | 涨跌幅 | 涨家数 | 跌家数 | 核心个股 | 资金流向 |
|------|------|--------|--------|--------|----------|----------|
| 1 | 白酒 | +3.37% | 19 | 0 | ... | ... |
（下面再列20行表格）
```

## 关键改进要点（v3新增，2026-05-27）

1. **30秒速读**：开头+结尾都有，30秒了解核心结论
2. **预测命中率**：x/y = z%，不是模糊的64/100分
3. **规则系统R-xx**：每条规则有触发条件+有效期+状态
4. **股票池五级分类**：核心/重点/简短/降级/凑数
5. **明日继承项**：明确哪些规则/判断继承到明天
6. **盘中策略验证**：如有盘中文件则验证
7. **主线独立跟踪**：重要方向不泛化在板块大类中
8. **数据源逐项标注**：每项数据来源+等级+状态
9. **潜力股选股**（晚间版专有）：不限观察池，从技术突破/消息催化/板块轮动/超跌反弹四维度选股
10. **预测验证闭环**：晚间预测→次日收盘验证→命中率→改进逻辑
