## 三、ETF观察池输出

```
【ETF 观察池】

A池：核心配置 ETF
（宽基、红利、债券、黄金等长期配置工具）

B池：战术轮动 ETF
（行业、主题、周期ETF，有明确催化）

C池：情绪样本 ETF
（高波动主题、短线炒作，只观察不配置）
```

## 五、ETF进日报格式

### 早报增加

```
【ETF 观察】
1. 宽基ETF：判断市场整体风险偏好
2. 行业ETF：判断主线板块持续性
3. 债券ETF：判断利率和防守资产
4. 黄金ETF：判断避险和实际利率
5. QDII ETF：判断海外市场与汇率
6. 商品ETF：判断通胀和周期
```

### 收盘复盘增加

```
【ETF 复盘】
1. 今日ETF强弱
2. 行业ETF是否印证题材
3. 宽基ETF是否强于个股赚钱效应
4. 债券/黄金ETF是否反映避险
5. QDII ETF是否有溢价风险
6. ETF模拟盘是否触发
7. ETF观察池是否调整
```
