# Cron 修改确认模板

修改任何cron任务前，必须先输出以下格式，等主人确认：

```
📋 Cron变更清单

任务：{任务名称} ({job_id})
变更类型：prompt / enabled_toolsets / schedule / context_from / 其他

变更内容：
- 旧值：{当前值}
- 新值：{目标值}

影响：
- 下次执行时间：{next_run_at}
- 预计token消耗变化：{增加/减少/不变}
- 风险：{可能的问题}

是否确认执行？
```

## 禁止事项
- 禁止批量修改多个cron任务而不逐个确认
- 禁止在报告生成过程中顺带修改cron
- 禁止在修复其他问题时"顺便"改cron配置
