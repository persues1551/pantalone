# 买入/卖出规则设计模式

> 从v1到v3.1的规则演进，提炼为可复用的设计模式

## 买入规则设计模式

### 1. 量化评分系统（满分100，门槛60）

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 板块强度 | 25分 | 当日板块涨幅>1%=20, >2%=25, <0%=0 |
| 技术位置 | 25分 | RSI 40-60=25, 60-70=20, >75=5 |
| 量价配合 | 20分 | 放量突破=20, 缩量上涨=10, 放量下跌=0 |
| 消息催化 | 15分 | 重大利好(+4)=15, 一般利好=10, 无消息=5 |
| 资金流向 | 15分 | 主力净流入>5000万=15, >0=10, 净流出=0 |

**仓位计算**：
- 60-69分 → 10%仓位（2万）
- 70-79分 → 15%仓位（3万）
- 80-89分 → 20%仓位（4万）
- 90+分 → 25%仓位（5万）

### 2. 分批建仓

| 条件 | 操作 |
|------|------|
| 评分≥60且大盘情绪≥5/10 | 第一批买入50%计划仓位 |
| 次日确认（不跌>2%） | 第二批买入剩余50% |
| 次日跌>2% | 取消第二批，观察 |

### 3. 大盘环境过滤器

| 大盘状态 | 判断标准 | 最大暴露 |
|----------|----------|----------|
| 🟢 主升浪 | 涨停>80 | 70% |
| 🟢 震荡偏强 | 涨停>50且创业板>0 | 50% |
| 🟡 震荡 | 涨停>30且创业板>-1 | 40% |
| 🔴 回调 | 创业板<-1.5 | 20% |
| ⚫ 恐慌 | 跌停>20 | 0% |

**⚠️ 关键修复**：涨停>80直接判定主升浪，不依赖创业板涨跌幅。首日无前日数据时会误判。

### 4. 高价股适配

```python
# ❌ 旧规则：高价股被5万上限误拦
max_per_stock = min(total_value * 0.25, 50000)

# ✅ 新规则：确保能买至少1手
min_lot_cost = price * 100
max_per_stock = max(total_value * 0.25, min_lot_cost)
```

### 5. 禁止买入条件（硬约束）

| 条件 | 理由 |
|------|------|
| RSI>75 | 超买区 |
| 连续3日缩量上涨 | 量价背离 |
| 高开5%以上 | 追高成本太高 |
| 涨停板打开 | 主力出货信号 |
| ST/退市风险 | 系统性风险 |
| 数据缺失>3项 | 无法判断 |

## 卖出规则设计模式

### 三层清仓逻辑

#### 第一层：均线趋势
| 条件 | 操作 |
|------|------|
| 跌破5日均线 | 减半仓 |
| 跌破10日均线 | 清仓 |
| 跌破5日后次日收回 | 不减仓（假破） |

#### 第二层：分批止盈
| 浮盈 | 操作 |
|------|------|
| +12% | 卖出1/3 |
| +20% | 再卖出1/3 |
| +30% | 剩余1/3跟踪止盈 |
| 回撤至最高点-5% | 卖出剩余 |

**⚠️ 关键修复**：+8%止盈太早，改为+12%给趋势空间。

#### 第三层：时间窗口
| 持仓天数 | 操作 |
|----------|------|
| 1-3天 | 正常持有 |
| 3-5天 | 关注止盈信号 |
| 5-7天 | 无条件减半仓 |
| 10天+ | 无条件清仓 |

### 综合判断流程（每日收盘后）

```
1. 股价是否跌破5日均线？→ 减半仓
2. 股价是否跌破10日均线？→ 清仓
3. 浮盈是否达到+12%/+20%/+30%？→ 分批止盈
4. 持仓是否超过5天？→ 关注止盈信号
5. 持仓是否超过10天？→ 无条件清仓
6. 板块资金是否连续2日流出？→ 减仓
以上任一条件触发即执行，不犹豫。
```

## 历史验证

| 版本 | 收益率 | 跑赢创业板 | 关键改进 |
|------|--------|-----------|----------|
| v1(旧规则) | -0.2% | -5.03pp | — |
| v2(回放) | +7.08% | +2.25pp | 已知结果倒推 |
| v3.0 | +2.31% | -2.52pp | 量化评分+三层清仓 |
| v3.1 | +5.15% | +0.32pp | 高价股适配+止盈放宽+涨停优先 |

## 反模式

1. **只有买入和止损，没有中间地带**：规则只有两个极端点，中间是真空
2. **止盈太早**：+8%就卖，错过趋势股后续涨幅
3. **不分批**：一次性买入/卖出，没有确认机制
4. **不看大盘**：不管市场环境，盲目执行买入
5. **高价股被误拦**：5万上限拦住评分100的高价股
6. **时间窗口缺失**：没有持仓天数限制，长期套牢
