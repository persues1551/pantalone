# OCIFQ选股框架详细参考

灵感来源：川沐（小红书5303101410）的行业印钞机框架。

## 核心公式

**行业印钞机 = 寡头定价权(O) × 长周期催化(C) × 行业利润断层(I) × 公司财务三爆(F) × 连续季报验证(Q)**

## 五维评分详解

### O — 寡头定价权（25%）
评估维度：
- 供应商数量（越少越集中越好）
- CR3/CR5集中度（行业前三/五市占率）
- 专利/认证壁垒
- 客户导入难度（切换成本）
- 提价能力（历史提价记录）
- 长期毛利率稳定性（≥30%为佳）

评分参考：
- 20-25分：绝对垄断（茅台、长江电力）
- 15-19分：强寡头（北方华创、宁德时代）
- 10-14分：有壁垒但竞争激烈（紫金矿业）
- <10分：代工/低壁垒（工业富联）

### C — 长周期催化（20%）
评估维度：
- 催化持续≥4个季度
- 来源：AI、能源转型、技术迭代、供需错配、政策、出海、库存周期
- 非短期题材炒作，需有产业逻辑支撑

评分参考：
- 17-20分：持续≥6季度的强催化（AI算力、半导体国产替代）
- 13-16分：持续4-5季度的中等催化（智能驾驶、储能）
- 9-12分：持续性不确定（消费复苏、政策刺激）
- <9分：短期题材（概念炒作）

### I — 行业利润断层（20%）
评估维度：
- 多家公司同步出现收入/利润/毛利率改善
- 上下游联动验证
- 是否属于行业首次从低利润进入高利润状态
- 高低端产品同步改善

评分参考：
- 17-20分：同行业≥3家公司同步暴增（半导体设备2026Q1）
- 13-16分：同行业2家公司同步改善（有色金属）
- 9-12分：行业改善但分化明显（电力）
- <9分：行业低迷或下行（光伏2026Q1）

### F — 公司财务三爆（25%）
硬指标（必须同时满足≥2项）：
- 收入同比≥30% 或 环比≥15%
- 扣非净利润同比≥50% 或 环比≥30%
- 毛利率同比提升≥5pct 或 环比提升≥3pct
- 现金流、应收、存货同步健康

评分参考：
- 21-25分：三爆全中（中际旭创2026Q1）
- 16-20分：二爆达标（宁德时代、新易盛）
- 11-15分：一爆达标或接近达标
- <11分：不达标

### Q — 连续季报验证（10%）
评估维度：
- 最近4个季度收入是否连续改善
- 最近4个季度利润是否连续改善
- 最近4个季度毛利率是否连续改善
- 核心产品线是否持续放量

评分参考：
- 9-10分：4季度连续改善且加速
- 7-8分：4季度连续改善但增速平稳
- 5-6分：3季度改善1季度持平
- <5分：有季度恶化

## 硬门槛（降级规则）

| 条件 | 最高评级 |
|------|----------|
| 三爆缺一项 | B级 |
| F低于70分 | 总分最高75 |
| O低于60分 | 最高A |
| C低于60分 | 最高B |
| I低于60分 | 最高A |
| Q低于50分 | 最高A |
| 利润来自一次性收益/补贴/资产处置/公允价值变动 | 单独扣分 |

## 评级体系

| 分数 | 等级 | 含义 |
|------|------|------|
| 90-100 | S级 | 高度确认，行业印钞机 |
| 85-89 | A+ | 核心候选，等待买点 |
| 78-84 | A | 强候选，值得跟踪 |
| 70-77 | B | 周期反转或题材增强 |
| <70 | 普通 | 阶段性题材，不推荐 |

## 双轨评分体系（v3.5）

```
综合评分 = OCIFQ选股(60%) + Pantalone择时(40%)

OCIFQ选股维度（60分）：
├─ O 寡头定价权（15%）
├─ C 长周期催化（10%）
├─ I 行业利润断层（10%）
├─ F 财务三爆（15%）
└─ Q 连续季报（10%）

Pantalone择时维度（40分）：
├─ RSI/均线位置（10%）
├─ 量价配合（8%）
├─ 资金流向（7%）
├─ 消息催化（8%）
└─ 板块强度（7%）
```
