# 继承项系统 (Inheritance System)

## 概念

每日收盘复盘生成「明日继承项」，明确哪些判断/规则/股票池状态需要继承到明天。次日早盘预案必须先读取继承项。

## 继承项模板

```markdown
# 明日继承项 (2026-xx-xx收盘)

## 市场状态继承
- 今日状态：{防守/进攻/震荡}
- R-xx：生效中（{触发条件}，有效期至{日期}）

## 股票池继承
### 核心标的
- xxx：{状态}

### 重点观察
- xxx：{状态}

### 降级复核
- xxx：{降级原因}

## 规则继承
- R-xx：{规则内容}
- R-xx：{规则内容}

## 关键判断继承
- xxx方向：{判断}
- xxx风险：{判断}

## 明日需验证项
- xxx：{需要验证什么}
```

## 使用流程

1. **收盘复盘**：生成继承项，写入 `~/.hermes/cache/amadeus/inheritance_YYYYMMDD.md`
2. **次日早盘**：读取继承项，作为预案的起点
3. **预案验证**：对比继承项中的判断与实际表现
4. **更新继承项**：收盘时生成新的继承项

## 文件位置

- 当前继承项：`~/.hermes/cache/amadeus/inheritance_latest.md`
- 历史继承项：`~/.hermes/cache/amadeus/inheritance_YYYYMMDD.md`
