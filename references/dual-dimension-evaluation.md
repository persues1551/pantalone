# 双维度评估框架（长线+短线）

## 核心理念
长线看好 = 基本面驱动，决定"值不值得买"
短线看好 = 技术面+情绪面驱动，决定"什么时候买"
好公司 + 好时机 = 好交易

## 长线评估（权重60%）
| 维度 | 权重 | 评估要点 |
|------|------|----------|
| 行业趋势 | 25% | 渗透率、政策支持、技术迭代 |
| 公司竞争力 | 25% | 市占率、技术壁垒、品牌溢价 |
| 业绩增长 | 20% | 营收增速、利润增速、毛利率 |
| 估值合理性 | 15% | PE/PB分位数、PEG、DCF空间 |
| 资产质量 | 15% | 现金流、负债率、ROE |

## 短线评估（权重40%）
| 维度 | 权重 | 评估要点 |
|------|------|----------|
| 技术形态 | 30% | 均线、MACD、RSI、布林带 |
| 资金流向 | 25% | 主力资金、北向资金、融资余额 |
| 市场情绪 | 20% | 板块热度、涨停效应、市场情绪 |
| 催化剂 | 15% | 事件、消息、政策、财报 |
| 量价配合 | 10% | 成交量、换手率、量比 |

## 四象限策略矩阵
```
                    短线看好
                    高(B+)
                     │
        ┌────────────┼────────────┐
        │   激进买入  │   核心持仓  │
长线高 ─┼────────────┼────────────┤─ 长线低
看好(B+)│   逢低布局  │   观察验证  │ 看好(D+)
        └────────────┼────────────┘
```

## 综合评分
综合分 = 长线分 × 60% + 短线分 × 40%

## 策略定义
| 象限 | 长线 | 短线 | 策略 | 仓位 |
|------|------|------|------|------|
| 核心持仓 | A/B | A/B | 重仓持有 | 30-50% |
| 激进买入 | A/B | A/B | 积极买入 | 20-30% |
| 逢低布局 | A/B | C/D | 等待时机 | 10-20% |
| 观察验证 | C | C | 小仓观察 | 5-10% |

## 评分更新命令
python3 pool_manager.py update CODE --long SCORE REASON --short SCORE REASON
