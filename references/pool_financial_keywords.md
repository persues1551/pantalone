# 财报关键词筛选框架

## 来源

知乎职业投资人分享的选股方法：
- 每年1月、4月、7月、10月是财报季
- 看业绩大增公司的财报、招股说明书
- 关注特定关键词，叠加业绩大增=重点关注

## 筛选逻辑

### 基础条件
1. 净利润增长>30%
2. 营收增长>10%
3. ROE>5%

### 关键词验证
在财报/交流/调研中检查是否包含上述关键词

### 叠加规则
- 业绩大增 + 关键词命中 = 重点关注
- 业绩大增 + 无关键词 = 普通关注
- 业绩一般 + 关键词命中 = 待观察

## 与OCIFQ整合

### F维度（财务三爆）增强
原F维度只看财务数据，现在增加关键词验证：

| 条件 | 得分 |
|------|------|
| 净利润增长>100% | +25分 |
| 净利润增长>50% | +20分 |
| 净利润增长>30% | +15分 |
| 营收增长>30% | +15分 |
| 营收增长>10% | +10分 |
| ROE>15% | +10分 |
| ROE>10% | +7分 |
| ROE>5% | +5分 |
| **关键词命中** | **+10分** |
