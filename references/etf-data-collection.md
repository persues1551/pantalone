# ETF 数据采集实操指南（2026-05-17实测）

## 问题

AKShare `fund_etf_hist_em` 批量获取ETF历史净值时频繁 `RemoteDisconnected`，19只全部失败。原因可能是东方财富反爬或连接池耗尽。

## 解决方案：东方财富 Web API 三件套

### 1. 实时行情（替代 fund_etf_spot_em）

```
URL: https://push2.eastmoney.com/api/qt/stock/get
参数:
  secid = 1.{code} (沪市51/56开头) 或 0.{code} (深市15开头)
  fields = f43,f44,f45,f46,f47,f48,f50,f57,f58,f60,f116,f117,f162,f167,f168,f169,f170,f171

返回字段:
  f43=最新价(×1000)  f44=最高  f45=最低  f46=开盘
  f47=成交量  f48=成交额  f170=涨跌幅(×100)  f168=换手率(×100)
  f116=总市值  f117=流通市值
```

### 2. 基金净值估算（替代 fund_etf_fund_info_em）

```
URL: https://fundgz.1234567.com.cn/js/{code}.js
返回: jsonpgz({"fundcode":"510300","name":"...","jzrq":"2026-05-14","dwjz":"4.9335","gsz":"4.8782","gztime":"2026-05-15 15:00"})

字段:
  dwjz = 单位净值 (NAV)
  gsz  = 估算净值 (IOPV估算)
  jzrq = 净值日期
  gztime = 估算时间
```

### 3. 基金详情（费率/跟踪指数/基金公司/成立日期）

```
URL: https://fund.eastmoney.com/pingzhongdata/{code}.js
返回: JavaScript变量
  fS_name = 基金名称
  fS_code = 基金代码

URL: https://fundf10.eastmoney.com/jbgk_{code}.html
返回: HTML页面，正则提取:
  管理费率.*?(\d+\.?\d*)%
  托管费率.*?(\d+\.?\d*)%
  跟踪指数.*?<a[^>]*>([^<]+)</a>
  基金管理人.*?<a[^>]*>([^<]+)</a>
  成立日期.*?(\d{4}-\d{2}-\d{2})
```

### 4. IOPV/折溢价（从 fund_etf_spot_em 获取）

AKShare `fund_etf_spot_em()` 单次调用可用（不循环），返回字段：
- `IOPV实时估值` — 基金份额参考净值
- `基金折价率` — 折溢价率（负=折价，正=溢价）

**折溢价计算**：`(市场价 - IOPV) / IOPV × 100%`

## 注意事项

1. `fundgz.1234567.com.cn` 部分QDII基金（513030德国、159981能源化工）返回空
2. `push2.eastmoney.com` 需要 `User-Agent` header
3. 所有API建议 0.3-0.5s 间隔请求，避免被封
4. 今日数据日期为T-1（如5月17日分析，可用数据截至5月15日收盘）
