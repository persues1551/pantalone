# 财报关键词筛选法（融入OCIFQ F维度）

## 来源
知乎职业投资人分享的高效选股方法（4835赞）。

## 核心逻辑
财报季（1/4/7/10月）→ 看业绩大增公司财报 → 关注关键词 → 双重验证入池

## 关键词库（10个）

| 关键词 | 含义 | 对应OCIFQ维度 |
|--------|------|---------------|
| 产品供不应求 | 需求>供给，有提价空间 | O（寡头定价权） |
| 行业高景气度上行 | 行业周期向上 | C（长周期催化） |
| 市场超预期拓展 | 新市场/新客户打开 | C（长周期催化） |
| 新品上市持续超预期 | 产品迭代成功 | F（财务三爆） |
| 产品价格中枢持续上涨 | 提价能力 | O（寡头定价权） |
| 供给偏紧 | 供给端约束 | I（行业利润断层） |
| 需求旺盛 | 需求端强劲 | I（行业利润断层） |
| 产能利用率提升 | 产能释放 | F（财务三爆） |
| 订单饱满 | 订单能见度高 | Q（连续季报） |
| 毛利率提升 | 盈利能力改善 | F（财务三爆） |

## OCIFQ F维度增强规则
原F维度标准：收入同比≥30%、扣非净利润同比≥50%、毛利率同比提升≥5pct
增强后：财报/交流/调研中出现≥2个关键词 → F维度+10分

## 评分公式
F维度得分 = 基础分（业绩增长） + 关键词加分（每个+2分，上限+10分）

## 数据来源
1. 业绩快报：ak.stock_yjkb_em(date) — 净利润增长、营收增长、ROE
2. 业绩预告：ak.stock_yjyg_em(date) — 预告类型、预告内容（关键词匹配）
3. 财报公告：巨潮公告API — 年报/季报正文

## Pitfall
1. 业绩快报数据量有限：AKShare每次只返回~20条，需多次调用不同季度
2. 关键词匹配需人工审核：自动匹配可能误报
3. ST股需排除：业绩大增但ST的股票风险高
4. 行业验证必要：单家公司业绩好不等于行业趋势，需I维度交叉验证
