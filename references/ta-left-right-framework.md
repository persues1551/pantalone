# 实用技术分析双框架（左侧+右侧）

来源：知乎投资讨论帖，经实战验证（2026-06-02）。

## 核心思路

> "说技术分析没用的，这种人你笑笑就好，然后远离。"

技术分析不需要复杂指标，两个最实用的方法：

## 方法一：右侧——放量突破

**规则**：看所在价格平台有没有放量。如果持续向上放量 → 干就完了。

**具体操作**：
1. 找到当前价格的整理平台（横盘区间）
2. 观察是否出现**连续3天以上放量突破**
3. 量能应逐日递增或至少不萎缩
4. 突破后回踩不破平台 → 确认有效

**适用场景**：趋势交易、动量策略、板块轮动追涨

**注意**：
- 放量必须是**持续的**，单日放量可能是诱多
- 结合RSI看是否超买（RSI>70谨慎）
- 板块共振更可靠（同板块多只同时放量）

## 方法二：左侧——均线+低PE+高股息

**规则**：
1. 看近10天平均换手率
   - 换手率 < 1% → 用 **MA120**
   - 换手率 ~ 2% → 用 **MA50**
2. 现价远低于参考均线 → 比大部分投资者筹码便宜
3. **前提条件**：PE要低、股息率要高

**具体操作**：
```
步骤1：计算近10日平均换手率
步骤2：选择参考均线（<1.5%用MA120，>=1.5%用MA50）
步骤3：计算距均线距离
步骤4：检查PE和股息率
步骤5：判断左侧机会等级
```

**左侧机会等级判定**：

| 距均线 | PE | 股息率 | 等级 | 操作 |
|--------|-----|--------|------|------|
| <-15% | <20 | >3% | ⭐⭐⭐⭐⭐ 强烈左侧 | 重仓买入 |
| <-15% | <20 | 1-3% | ⭐⭐⭐⭐ 左侧机会 | 分批买入 |
| <-10% | <30 | >2% | ⭐⭐⭐ 较好左侧 | 试探性买入 |
| <-10% | >50 | <1% | ❌ 不满足前提 | 不买（PE太高） |
| >+10% | 任意 | 任意 | ❌ 远高于均线 | 不追涨 |

## 与OCIFQ的结合

OCIFQ选股（基本面70%）+ 技术分析（择时30%）：
- OCIFQ选出好公司 → 用左侧方法找买点
- 右侧方法用于确认趋势 → 配合Pantalone择时维度
- **核心原则**：技术分析是辅助，不是灵魂。基本面才是。

## 数据获取

计算MA需要历史K线数据。**腾讯实时API不提供MA数据**（字段6-9是成交量不是MA）。

正确方法：用新浪K线API获取历史数据后计算：
```bash
curl -s "https://quotes.sina.cn/cn/api/jsonp_v2.php/var/CN_MarketDataService.getKLineData?symbol=sh601899&scale=240&ma=no&datalen=130"
```
返回JSON数组，包含day/open/high/low/close/volume。取close字段计算MA5/10/20/50/120。

## Pitfalls

1. **左侧策略前提不能忽略**：低于MA但PE>50的股票不是左侧机会，是价值陷阱
2. **换手率分界线不是绝对的**：1%/2%是参考值，实际应根据流通盘大小调整
3. **MA120需要120天数据**：新股/次新股数据不足时降级用MA60或MA50
4. **ETF不适用左侧方法**：ETF没有PE/股息率概念，应看趋势和估值分位
5. **单日换手率可能失真**：用近10日平均换手率更稳定
