# 规则系统 (Rule System)

## 概念

规则系统是收盘复盘的核心机制。每条规则有：
- **编号**：R-xx（递增）
- **触发条件**：什么情况下触发
- **有效期**：何时过期
- **状态**：生效中/已过期/已取消
- **继承性**：是否自动继承到明日

## 规则模板

```markdown
## R-xx: {规则名称}
- **触发条件**：{具体条件}
- **生效时间**：{yyyy-mm-dd HH:MM}
- **有效期**：{永久/至yyyy-mm-dd HH:MM/触发后xx小时}
- **状态**：生效中
- **继承性**：是/否
- **执行动作**：{具体做什么}
```

## 规则生命周期

1. **创建**：收盘复盘中发现需要新增规则
2. **生效**：写入规则系统，标记触发条件和有效期
3. **继承**：每日收盘复盘检查哪些规则需要继承到明日
4. **过期**：有效期到达后自动标记为已过期
5. **取消**：条件不再适用时手动取消

## 使用方式

1. 收盘复盘时检查当前规则状态
2. 新增规则写入 `rules/active_rules.md`
3. 过期规则移入 `rules/expired_rules.md`
4. 继承项写入 `rules/inheritance.md`
