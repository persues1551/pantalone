# 观察池哲学与规则 v2.0

## 核心定位

观察池不是"热点收藏夹"，而是**逻辑实验场**。

```
选股逻辑假设 → 入池观察 → 验证结果 → 修正逻辑 → 复利积累
```

**目的**：通过持续验证，建立可复制的选股逻辑体系。

---

## 一、入池规则

### 1.1 入池必须满足（缺一不可）

| 条件 | 说明 | 示例 |
|------|------|------|
| **逻辑假设** | 为什么选它？预期什么？ | "AI算力需求爆发→光模块业绩兑现" |
| **验证时间** | 什么时候验证？ | "1个月后看Q2财报" |
| **成功标准** | 什么算逻辑兑现？ | "营收增速>30%，毛利率提升" |
| **失败标准** | 什么算逻辑证伪？ | "营收增速<10%，或行业需求下降" |

### 1.2 入池逻辑分类

| 逻辑类型 | 说明 | 验证周期 |
|----------|------|----------|
| **业绩驱动** | 预期财报超预期 | 季报后 |
| **行业周期** | 行业景气度上行 | 1-3个月 |
| **政策催化** | 政策利好落地 | 1-2个月 |
| **技术突破** | 新技术/新产品放量 | 2-3个月 |
| **估值修复** | 低估+催化剂 | 1-2个月 |
| **事件驱动** | 并购/重组/订单 | 事件落地后 |

## 二、长短线分离

### 2.1 长线池（A/B池）

**侧重**：基本面、估值、行业趋势
**验证周期**：30天
**策略**：长线持有

**筛选逻辑**：
1. 财报三爆（F维度）：净利润增长>30%, 营收增长>10%, ROE>5%
2. 估值合理（V维度）：PE<50, PB<15
3. 行业景气（I维度）：热门行业
4. 竞争格局（C维度）：龙头优先（市值>100亿 或 ROE>15%）

### 2.2 短线池（C池）

**侧重**：技术面、资金、情绪
**验证周期**：1天
**策略**：短线交易

**筛选逻辑**：
1. 涨停股
2. 成交额>1亿
3. 换手率5%-30%
4. 排除ST股

---

## 三、验证规则

### 3.1 验证结果分类

| 结果 | 定义 | 处理 |
|------|------|------|
| ✅ 逻辑兑现 | 成功标准达成 | 升级池子 |
| ⚠️ 逻辑偏移 | 部分达成 | 延长验证 |
| ❌ 逻辑证伪 | 失败标准达成 | 退池 |

### 3.2 验证节奏

- **长线池**：30天验证一次
- **短线池**：每天验证

## 五、5次迭代优化框架

筛选标准需迭代优化，每次迭代记录淘汰原因：

```
第1次：财报三爆 → 基础条件
第2次：估值合理 → PE/PB筛选
第3次：行业景气 → 热门行业
第4次：竞争格局 → 龙头优先
第5次：综合评分 → 排序入池
```
