# 报告模板v2（量化增强版）说明

## 模板清单

| 模板 | 文件 | 主要改进 |
|------|------|---------|
| **早盘预案** | `templates/morning_template.md` | +量化信号层+宏观数据层 |
| **午间复盘** | `templates/midday_template.md` | +量化信号更新+因子追踪 |
| **收盘复盘** | `templates/review_template.md` | +回测验证+因子轮动+ML评估 |

## 五次迭代改进

### 第一次：审视现有问题
- 现有模板是主观交易者框架，缺少量化投资核心要素
- 缺少ML信号、因子分析、宏观数据、技术指标量化

### 第二次：添加量化信号层
- ML模型信号（LightGBM/XGBoost评分）
- 因子信号（IC值、等级、多空收益）
- 技术指标量化（RSI/MACD/布林带位置）
- 量化信号汇总（加权评分）

### 第三次：添加宏观数据层
- 全球市场（美股/港股/汇率）
- 商品价格（原油/黄金/铜）
- 宏观事件日历

### 第四次：添加回测验证层
- ML信号回测（准确率、平均收益）
- 因子表现回测（IC、多空收益、因子轮动）
- 技术指标验证（预测准确度）
- 量化信号综合评估

### 第五次：添加因子追踪层
- 因子有效性追踪
- 因子轮动信号
- 量化信号改进建议

## 量化信号权重

| 信号维度 | 权重 | 说明 |
|----------|------|------|
| ML信号 | 30% | LightGBM/XGBoost评分 |
| 因子信号 | 25% | IC值、因子等级 |
| 技术面 | 20% | RSI/MACD/布林带/ATR |
| 情绪面 | 15% | 涨跌停/封板率/连板 |
| 资金面 | 10% | 板块资金流/北向 |

**综合评分阈值**：>70积极，50-70中性，<50谨慎

## 使用流程

```bash
# 早盘：获取量化信号
python3 ~/.hermes/scripts/amadeus/pantalone_tools_hub.py --comprehensive --stock 600519

# 午盘：更新信号，验证早盘信号准确度
# 收盘：回测验证，统计准确率，生成改进建议
```

## 专业投资人视角改进

| 维度 | 改进前 | 改进后 |
|------|--------|--------|
| **信号来源** | 主观判断 | 主观+量化（ML/因子/技术） |
| **数据维度** | 价格+情绪 | +宏观+资金流+技术指标量化 |
| **验证方式** | 定性验证 | +IC分析/回测/因子有效性 |
| **风险度量** | 主观风控 | +VaR/夏普/最大回撤 |
| **因子追踪** | 无 | +因子表现+因子轮动 |
