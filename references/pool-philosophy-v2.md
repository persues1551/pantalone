# 观察池哲学与规则 v2.0

## 核心定位

观察池不是"热点收藏夹"，而是**逻辑实验场**。

```
选股逻辑假设 → 入池观察 → 验证结果 → 修正逻辑 → 复利积累
```

**目的**：通过持续验证，建立可复制的选股逻辑体系。

---

## 一、入池规则（缺一不可）

| 条件 | 说明 |
|------|------|
| **逻辑假设** | 为什么选它？预期什么？ |
| **验证时间** | 什么时候验证？短线=1天 |
| **成功标准** | 什么算逻辑兑现？ |
| **失败标准** | 什么算逻辑证伪？ |

### 入池逻辑分类

| 逻辑类型 | 验证周期 |
|----------|----------|
| **业绩驱动** | 1天（短线）/ 季报后（长线） |
| **行业周期** | 1-3天 |
| **政策催化** | 1-2天 |
| **技术突破** | 1-3天 |
| **事件驱动** | 1天 |

## 二、双维度评估（长线+短线）

**长线看好**（60%权重）= 基本面驱动，决定"值不值得买"
- 行业趋势（25%）、公司竞争力（25%）、业绩增长（20%）、估值合理性（15%）、资产质量（15%）

**短线看好**（40%权重）= 技术面+情绪面，决定"什么时候买"
- 技术形态（30%）、资金流向（25%）、市场情绪（20%）、催化剂（15%）、量价配合（10%）

**综合分 = 长线分 × 60% + 短线分 × 40%**

### 四象限策略

| 象限 | 长线 | 短线 | 策略 | 仓位 |
|------|------|------|------|------|
| **核心持仓** | A/B | A/B | 重仓持有 | 30-50% |
| **激进买入** | A/B | A/B | 积极买入 | 20-30% |
| **逢低布局** | A/B | C/D | 等待时机 | 10-20% |
| **观察验证** | C | C | 小仓观察 | 5-10% |

## 三、财报关键词筛选（知乎职业投资人方法）

每年1月、4月、7月、10月是财报季。

**关键词**：产品供不应求、行业高景气度上行、市场超预期拓展、新品上市持续超预期、产品价格中枢持续上涨、供给偏紧、需求旺盛、产能利用率提升、订单饱满、毛利率提升

**筛选逻辑**：业绩大增 + 关键词命中 → 重点关注

## 五、管理脚本

**统一脚本**：`scripts/amadeus/pool_manager.py`（v3.0）

```bash
python3 pool_manager.py status      # 查看状态
python3 pool_manager.py scores      # 查看评分排名
python3 pool_manager.py verify      # 检查待验证
python3 pool_manager.py sync        # 同步pool_state.json
python3 pool_manager.py add ...     # 入池
python3 pool_manager.py update ...  # 更新评分
python3 pool_manager.py remove ...  # 退池
```

**自动扫描**：`scripts/amadeus/pool_auto_scanner.py`（v1.2）

```bash
python3 pool_auto_scanner.py --max 10           # 全量扫描
python3 pool_auto_scanner.py --financial-only    # 只扫描财报
python3 pool_auto_scanner.py --ml-only           # 只扫描ML
python3 pool_auto_scanner.py --industry-only     # 只扫描行业
python3 pool_auto_scanner.py --news-only         # 只扫描新闻
```
