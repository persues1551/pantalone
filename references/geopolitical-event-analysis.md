# 地缘政治事件→受益标的分析框架

> 案例来源：2026-05-13 特朗普访华（16位商业巨头随行）

## 分析流程（6步）

### Step 1: 确认事件基本面
- 时间、性质、参与方、历史意义
- 数据源：新浪首页 `grep` 关键词 → 逐篇抓正文

### Step 2: 提取关键利益方
| 利益方类型 | 示例 |
|-----------|------|
| 随行企业 | 特斯拉(马斯克)、苹果(库克)、高通、美光、波音、嘉吉 |
| 议题领域 | 贸易关税、半导体管控、农业采购、航空订单、金融开放 |
| 已释放信号 | 关税暂缓、采购承诺、管控松动预期 |

### Step 3: 按主题映射A股标的
每个议题方向独立分析，不混在一起：

| 主题 | 逻辑链条 | 典型标的 |
|------|---------|---------|
| 供应链直接受益 | 随行企业→其中国供应商 | 立讯(苹果)、三花(特斯拉) |
| 传统贸易筹码 | 历来谈判中用于交换的领域 | 北大荒(农产品)、东航(波音) |
| 管控松动预期 | 随行企业所在行业的出口限制 | 澜起/紫光(半导体) |
| 金融开放 | 华尔街巨头随行→互联互通 | 中信证券(券商) |
| 能源合作 | 能源贸易扩大 | 中石化(LNG) |

### Step 4: 分层确定性
- ⭐⭐⭐⭐⭐ 供应链直接受益（订单逻辑最硬）
- ⭐⭐⭐⭐ 传统筹码（历史惯例但非必然）
- ⭐⭐⭐ 管控松动（高弹性但高分歧）
- ⭐⭐ 金融/能源开放（长线逻辑）

### Step 5: 对比当日市场反应
- 事件日开盘走势验证市场预期
- 若利好方向反而低开 → 市场在博弈，不追
- 案例：2026-05-13 半导体低开（费城半导体隔夜跌3%），市场对管控松动预期不高

### Step 6: 加入观察池并设触发条件
- B池：确定性高的供应链标的
- C池：弹性大但分歧大的标的
- 每只必须有明确触发条件（如"苹果扩产信号""芯片管控松动"）
- 不设触发条件的不入池（防止无脑跟风）

## 注意事项
1. **利好兑现风险**：消息已发酵一段时间的标的可能已price in
2. **行程期间突发消息**：3天行程可能有意外进展，需持续跟踪
3. **不追分歧方向**：市场已经用脚投票的方向（如低开的半导体），等确认信号
4. **美股分析需yfinance**：`pip install yfinance --break-system-packages`，否则只能靠新闻推断
