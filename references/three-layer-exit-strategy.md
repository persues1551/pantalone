# 三层清仓逻辑详细规则（2026-05-14 建立）

## 背景

原清仓逻辑只有两个极端触发器：止损-5%和涨停骤降。中间地带完全真空，导致：
- 新易盛持仓仅1天就卖出（涨停骤降触发），少赚约1500元
- 工业富联持仓7天未止盈，从+12%回吐到+8%
- 立讯精密持仓3天未止盈，从+5.3%变成-0.5%

## 三层判断体系

### 第一层：趋势跟踪（均线系统）

| 条件 | 操作 | 理由 |
|------|------|------|
| 股价站稳5日均线上方 | 持有不动 | 短期趋势完好 |
| 跌破5日均线 | 减半仓 | 短期趋势转弱 |
| 跌破10日均线 | 清仓 | 中期趋势转弱 |
| 跌破5日后次日收回 | 不减仓（假破） | 避免洗盘被震出 |

**5日均线计算**：取最近5个交易日收盘价的算术平均值。

### 第二层：分批止盈（锁定利润）

| 浮盈 | 操作 | 理由 |
|------|------|------|
| +8% | 卖出1/3 | 锁定部分利润 |
| +15% | 再卖出1/3 | 大部分利润已锁定 |
| +25% | 剩余1/3跟踪止盈 | 让利润奔跑 |
| 回撤至最高点-5% | 卖出剩余 | 趋势结束信号 |

**注意**：分批止盈按股数取整至100股。若持仓仅100股，+8%时卖出全部。

### 第三层：时间窗口（匹配板块轮动）

| 持仓天数 | 操作 | 理由 |
|----------|------|------|
| 1-3天 | 正常持有 | 板块轮动周期内 |
| 3-5天 | 开始关注止盈信号 | 轮动窗口即将关闭 |
| 5-7天 | 无条件减半仓 | 超过轮动周期 |
| 10天+ | 无条件清仓 | 避免长期套牢 |

**依据**：5/6-5/14回放数据显示A股板块轮动周期约2-3天，主线个股最大涨幅通常在3-5天内完成。

### 综合判断流程（每日收盘后执行）

```
1. 股价是否跌破5日均线？→ 减半仓
2. 股价是否跌破10日均线？→ 清仓
3. 浮盈是否达到+8%/+15%/+25%？→ 分批止盈
4. 持仓是否超过5天？→ 关注止盈信号
5. 持仓是否超过10天？→ 无条件清仓
6. 板块资金是否连续2日流出？→ 减仓
以上任一条件触发即执行，不犹豫。
```
