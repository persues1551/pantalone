# 模拟盘交易日志规范

## 交易记录格式

```markdown
## 日期（星期）

**市场**：上证+X% 深证+X% 创业板+X%，涨停X只
**情绪**：X/10
**决策**：简述操作理由

| 操作 | 代码 | 名称 | 价格 | 股数 | 金额 | 理由 |
|------|------|------|------|------|------|------|
| 买入 | 600900 | 长江电力 | 27.00 | 200 | 5,400 | 高股息防御 |

**账户**：总资产XXX元 | 持仓X只 | 现金XXX元 | 风险暴露X% | 盈亏+XXX元(+X%)
```

## 交易规则

1. **成交价**：收盘价（保守原则）
2. **股数**：100股整数倍
3. **单票金额**：10,000-25,000元
4. **总风险暴露**：≤60%
5. **止损**：B池-5%，A池-8%

## 数据库操作

```bash
# 查看状态
python3 ~/.hermes/scripts/amadeus/amadeus_sim_integrate.py status

# 记录交易
python3 ~/.hermes/scripts/amadeus/amadeus_sim_integrate.py record '{"action":"buy","code":"600900","name":"长江电力","pool":"A","price":27.0,"shares":200,"reason":"模拟买入"}'

# 更新日盈亏
python3 ~/.hermes/scripts/amadeus/amadeus_sim_integrate.py daily_update
```
