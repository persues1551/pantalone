# 情绪温度验证规范

## 公式

```
情绪温度 = 涨停跌停比×25 + 市场宽度×25 + 连板高度×20 + 炸板率×15 + 成交额变化×15
```

## 分级

| 分数 | 分级 | 含义 |
|------|------|------|
| 0-20 | 冰点 | 10%机会 |
| 21-40 | 修复 | 30%机会 |
| 41-60 | 升温 | 60%机会 |
| 61-80 | 高潮 | 40%风险 |
| 81-100 | 退潮 | 20%风险 |

## 计算方法

### 涨停跌停比（权重25）
```
得分 = min(涨停数/跌停数, 50) / 50 * 25
```

### 市场宽度（权重25）
```
得分 = 上涨家数/(上涨家数+下跌家数) * 25
```

### 连板高度（权重20）
```
得分 = min(最高连板数, 10) / 10 * 20
```

### 炸板率（权重15）
```
得分 = (1 - 炸板数/(涨停数+炸板数)) * 15
```

### 成交额变化（权重15）
```
得分 = min(今日成交额/昨日成交额, 1.5) / 1.5 * 15
```

## 预测应用

1. **温度>70**：警惕回调，不追高
2. **温度<30**：可能反弹，关注低吸机会
3. **温度骤降**：退潮信号，减仓避险
4. **温度骤升**：情绪回暖，可加仓
